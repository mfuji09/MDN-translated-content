Sumário

O elemento HTML quebra-de-linha <br> produz uma quebra de linha em um texto (carriage-return).É útil para escrever poemas ou um endereço, onde a divisão de linha é significante.

Não use <br> para aumentar o espaço entre as linhas de texto; para isso use a propiedade CSS {{cssxref('margin')}} ou o elemento {{HTMLElement("p")}}.

Categorias de Conteúdo Conteúdo de Fluxo, conteúdo fraseado.
Conteúdo permitido Nenhum, é um {{Glossary("empty element")}}.
Omissão de tag Deve ter uma tag de abertura, e não deve ter uma tag de fechamento. Em documentos XHTML , escreve esse elemento como <br />.
Elementos pai permitidos Qualquer elemento que aceita conteúdo fraseado.
Interface DOM {{domxref("HTMLBRElement")}}

Atributos

Este elemento inclue os Atributos globais.

{{htmlattrdef("clear")}} {{deprecatedGeneric('inline','HTML4.01')}} {{obsoleteGeneric('inline','HTML5')}}
Indica onde começar a próxima linha depois da quebra.

Nota de Uso: Este atributo está obsoleto em {{HTMLVersionInline(5)}} e Não deve ser usado por autores. Use a propiedade CSS {{CSSxref('clear')}} em vez disso.

Exemplo

Mozilla Foundation<br>
1981 Landings Drive<br>
Building K<br>
Mountain View, CA 94043-0801<br>
USA

Mountain View, CA 94043-0801
