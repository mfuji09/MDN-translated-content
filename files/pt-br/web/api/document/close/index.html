{{APIRef("DOM")}}

O metodo document.close() termina a gravação em um documento, aberto com document.open().

Sintaxe

document.close();

Exemplo

// abre um documento e escreve nele.
// escreve o conteúdo no documento.
// fecha o documento.
document.open();
document.write("<p>O primeiro e unico conteúdo.</p>");
document.close();
