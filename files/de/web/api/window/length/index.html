Liefert die Anzahl an Frames ({{HTMLElement("frame")}} oder {{HTMLElement("iframe")}} elements) im aktuellen Fenster.

Syntax

framesCount = window.length;
  • framesCount ist die Anzahl an Frames.

Beispiel

if (window.length) {
  // this is a document with subframes
}
