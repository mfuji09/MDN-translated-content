**`::-moz-page`** [CSS](/zh-CN/docs/Web/CSS) [伪元素](/zh-CN/docs/Web/CSS/Pseudo-elements)是用于表示打印或打印预览时的一个单独页面的 [Mozilla 扩展](/zh-CN/docs/Web/CSS/Mozilla_Extensions)。

## 语法

```css
::-moz-page {
  /* ... */
}
```

## 规范

不属于任何规范。
