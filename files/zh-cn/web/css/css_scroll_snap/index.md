**CSS 滚动快照**是 CSS 的模块，它引入滚动捕捉位置，它强制滚动位置，即 {{Glossary("滚动容器")}}的 {{Glossary("滚动端口")}} 在滚动操作完成后可能结束的滚动位置。

> **备注：** 此模块的早期版本称为滚动捕捉点，已被弃用。CSS 滚动快照是当前实现。
