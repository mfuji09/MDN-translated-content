**`Crypto.subtle`** 是一个只读属性，返回一个 {{domxref("SubtleCrypto")}} 对象，可以用来做低级加密运算。

## 语法

```plain
var crypto = crypto.subtle;
```

### 值

一个可用于与 Web Crypto API 的低级加密功能交互的 {{domxref("SubtleCrypto")}} 对象。
