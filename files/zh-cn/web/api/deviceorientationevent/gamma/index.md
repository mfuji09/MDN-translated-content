{{ apiref("Device Orientation Events") }}

返回设备旋转时 Y 轴的值;即，多少度，介于之间`-90`和`90`，通过该装置被接通向左或向右。见[方向和运动数据解释](/en/DOM/Orientation_and_motion_data_explained)的细节。

## Syntax

```plain
var gamma = orientationEvent.gamma;
```
