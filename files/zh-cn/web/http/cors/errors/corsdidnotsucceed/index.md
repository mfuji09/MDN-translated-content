## 原因

```plain
原因：CORS 请求未成功
```

## 哪里错了？

使用 CORS 的{{Glossary("HTTP")}} 请求失败，因为 HTTP 连接在网络或协议级别失败。该错误与 CORS 没有直接关系，而是某种基本的网络错误。很多情况下，它是某个浏览器插件（比如广告拦截或隐私保护插件）阻止了请求而引起的。
