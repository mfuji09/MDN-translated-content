**Size**属性将会返回{{jsxref("Set")}}对象中元素的个数。

{{EmbedInteractiveExample("pages/js/set-prototype-size.html")}}

## 描述

`size`的值是一个整数，表示`Set`对象有多少条目。`size`的集合访问函数是`undefined`; 你不能改变这个属性。

## 例子

### 使用`size`

```js
var mySet = new Set();
mySet.add(1);
mySet.add(5);
mySet.add("some text")

mySet.size; // 3
```
