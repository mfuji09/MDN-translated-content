## 概述

**`Number.parseFloat()`** 方法可以把一个字符串解析成浮点数。该方法与全局的 {{jsxref("Global_Objects/parseFloat", "parseFloat()")}} 函数相同，并且处于 ECMAScript 6 规范中（用于全局变量的模块化）。

## 语法

```plain
Number.parseFloat(string)
```

### 参数

- `string`
  - : 被解析的字符串。

### 返回值

给定值被解析成浮点数，如果无法被解析成浮点数，则返回`NaN`
