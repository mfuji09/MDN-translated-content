## 概述

**`Number.NaN`** 表示“非数字”（Not-A-Number）。和 {{jsxref("Global_Objects/NaN", "NaN")}} 相同。

不必创建一个 {{jsxref("Global_Objects/Number", "Number")}} 实例来访问该属性，使用 `Number.NaN` 来访问该静态属性。
