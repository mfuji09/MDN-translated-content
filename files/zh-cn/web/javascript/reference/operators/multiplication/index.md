乘法运算符 (`*`) 计算操作数的乘积。

{{EmbedInteractiveExample("pages/js/expressions-multiplication.html")}}

## 语法

```plain
Operator: x * y
```

## 例子

### 使用数字相乘

```js
 2 * 2      // 4
-2 * 2     // -4
```

### 使用 Infinity 相乘

```js
Infinity * 0         // NaN
Infinity * Infinity  // Infinity
```

### 使用非数字相乘

```js
'foo' * 2 // NaN
```
