自减运算符 (`--`) 将它的操作数减一，然后返回操作数。

{{EmbedInteractiveExample("pages/js/expressions-decrement.html")}}

## <br>语法

```js
x--
--x
```

## 语法细节

如果使用后缀式，即将操作符放在操作数的后面 (如，`x--`)，运算会减一，然后返回减一之前的值。

如果使用前缀式，即将操作符放在操作数的前面 (如，`--x`)，运算会减一，然后返回减一之后的值。

## 示例

### 后缀式

```js
let x = 3;
y = x--;

// y = 3
// x = 2
```

### 前缀式

```js
let a = 2;
b = --a;

// a = 1
// b = 1
```
