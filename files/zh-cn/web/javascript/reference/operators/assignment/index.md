简单赋值操作符 (=) 用于为变量赋值。赋值表达式本身的值为要赋值的值。为了给多个变量赋一个值，可以链式使用赋值操作符。

{{EmbedInteractiveExample("pages/js/expressions-assignment.html")}}

## 语法

```plain
Operator: x = y
```

## 示例

### 简单赋值和链式赋值

```js
// 假设已经存在以下变量
//  x = 5
//  y = 10
//  z = 25

x = y     // x 为 10
x = y = z // x, y 都为 25
```
