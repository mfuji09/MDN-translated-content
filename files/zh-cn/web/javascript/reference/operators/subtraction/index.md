减法运算符（`-`）将两个操作数相减，并产生两者之差。

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

## 语法

```js-nolint
x - y
```

## 示例

### 数值减法

```js
5 - 3     // 2
3 - 5     // -2
```

### 非数值减法

```js
'foo' - 3 // NaN
```
