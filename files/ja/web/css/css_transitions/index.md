<p><ruby><strong>CSS トランジション</strong><rp> (</rp><rt>CSS Transition</rt><rp>) </rp></ruby>は CSS のモジュールの一つで、特定の CSS プロパティの値の間でスムーズな変化を作り出す方法を定義します。タイミング関数を使って、ただ作り出すだけでなく、その進化を定義することもできます。</p>
