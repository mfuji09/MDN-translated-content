<p><ruby><strong>CSS 書字方向</strong><rp> (</rp><rt>CSS Writing Modes</rt><rp>) </rp></ruby>は CSS モジュールの一つで、様々な国の書字方向、例えば左書き (ラテン系やインド語系の文章)、右書き (ヘブライ語やアラビア語の文章)、双方向 (左書きと右書きが混在する場合)、縦書き (一部のアジア言語で使われます) などを定義します。</p>
