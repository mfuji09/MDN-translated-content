概要

method は、 フォーム送信時に使用する HTTPメソッドを取得 / 設定する。

構文

string = form.method; //取得
form.method = string; //設定

例

var myform = document.forms["myform"];

myform.method = "post";

仕様

  • DOM Level 2 HTML: method
