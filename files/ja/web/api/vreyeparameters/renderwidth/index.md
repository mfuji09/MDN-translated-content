{{domxref("VREyeParameters")}} インターフェイスの **`renderWidth`** 読取専用プロパティは，各眼のビューポートの推奨レンダーターゲット幅をピクセル値で表します．

## シンタックス

```js
var curRenderWidth = myVRDisplay.getEyeParameters('left').renderWidth;
```

### 値

幅を表す数値．ピクセル単位．
