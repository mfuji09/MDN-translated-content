{{domxref("VREyeParameters")}} インターフェイスの **`renderHeight`** 読取専用プロパティは，各眼のビューポートの推奨レンダーターゲット高さをピクセル値で表します．

## シンタックス

```js
var curRenderHeight = myVRDisplay.getEyeParameters('left').renderHeight;
```

### 値

高さを表す数値．ピクセル単位．
