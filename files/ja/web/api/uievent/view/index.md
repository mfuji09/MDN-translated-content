{{APIRef("UI Events")}}

**`UIEvent.view`** は読み取り専用のプロパティで、イベントを生成したものの {{glossary("WindowProxy")}} オブジェクトを返します。ブラウザーでは、これはイベントが発行された {{ domxref("Window") }} です。

## 値

- `view` は `AbstractView` オブジェクトの参照です。
