{{APIRef("HTML DOM")}}

**`willValidate`** は {{domxref("HTMLObjectElement")}} インターフェイスの読み取り専用プロパティで、その要素が制約検証の候補であるかどうかを示す論理値を返します。 `HTMLObjectElement` オブジェクトの場合は常に `false` です。

## 値

A boolean value.
