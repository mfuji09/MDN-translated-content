**`CSSUnparsedValue.keys()`** メソッドは、配列内のそれぞれの添字のキーを持つ新しい **`Array Iterator`** オブジェクトを返します。

## 構文

```
CSSUnparsedValue.keys()
```

### 引数

なし。

### 返値

新しい {{jsxref("Array")}}。
