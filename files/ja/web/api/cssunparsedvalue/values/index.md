**`CSSUnparsedValue.values()`** メソッドは、 CSSUnparsedValue オブジェクト内のそれぞれの添字の値を持つ新しい **`Array Iterator`** オブジェクトを返します。

## 構文

```
CSSUnparsedValue.values()
```

### 引数

なし。

### 返値

新しい {{jsxref("Array")}}。
