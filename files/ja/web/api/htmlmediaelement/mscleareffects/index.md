**msClearEffects`** は [_HTMLMediaElement_](/ja/docs/Web/API/HTMLMediaElement) のメソッドで、 Internet Explorer と Microsoft Edge に固有の独自仕様です。

**msClearEffects`** は、メディアパイプラインからすべてのエフェクトをクリアします。

### 構文

```js-nolint
msClearEffects()
```

### 引数

このメソッドには引数がありません。

### 返値

このメソッドは値を返しません。
