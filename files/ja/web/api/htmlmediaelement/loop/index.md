{{APIRef("HTML DOM")}}

**`HTMLMediaElement.loop`** プロパティは、 HTML の {{htmlattrxref("loop", "video")}} 属性を反映し、メディア要素が最後に到達したときに最初からやり直すかどうかを制御します。

## 値

論理値です。

## 例

```js
var obj = document.createElement('video');
obj.loop = true; // true
```
