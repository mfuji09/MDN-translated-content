{{ApiRef("HTML DOM")}}

**`HTMLAnchorElement.hostname`** プロパティは、URL のドメインを含む文字列です。

## 値

文字列です。

## 例

```js
// <a id="myAnchor" href="/en-US/docs/HTMLAnchorElement"> 要素が文書内にあるとします
const anchor = document.getElementById("myAnchor");
anchor.hostname; // 'developer.mozilla.org' を返す
```
