**`prerender`** キーワードを {{HTMLElement("link")}} 要素の {{HTMLAttrxRef("rel", "link")}} 属性に設定すると、ユーザーが次の操作でターゲットとなるリソースを必要とする可能性があることをブラウザーに示唆します。したがって、ブラウザーはリソースの取得や処理を先取りすることで、ユーザーの操作性を向上させることができる可能性があります。
