ceiling 関数は小数を評価し、その小数以上の最も小さい整数を返します。

構文

ceiling(number )

引数

number
評価する数値。

返値

number 以上の最も近い整数。

例:

ceiling (5.2) = 6

ceiling (-5.2) = -5

定義

XPath 1.0 4.4

Gecko での対応

対応済み。
