--- title: Grid container slug: Glossary/grid_container tags: - CSS - Glossary - grid translation_of: Glossary/Grid_Container ---

要素に grid または inline-grid を使用すると、CSS グリッドレイアウト のグリッドコンテナーになります。そして直下の子要素はグリッドアイテムになります。

要素がグリッドコンテナーになると、グリッド整形文脈を形成します。グリッドコンテナー直下の子要素は、{{cssxref("grid-template-columns")}} や {{cssxref("grid-template-rows")}}を使って明示的に定義されたグリッドや、明示的に定義されたグリッドの外側に置かれた時には暗黙的に形成されるグリッドに沿って、配置されます。
