--- title: beacon (ビーコン) slug: Glossary/beacon translation_of: Glossary/beacon ---

Web ビーコンは、マークアップに埋め込まれた 1 ピクセル gif などの小さなオブジェクトで、Web サーバーまたはサードパーティのサーバーに情報を戻すために使用されます。

ビーコンは通常、統計目的でユーザーに関する情報を提供するために含まれています。

多くの場合、ビーコンは、ユーザーデータ、パフォーマンスメトリック、およびエラーレポートを収集するためのサードパーティのスクリプトに含まれています。

W3C Draft Beacon Specification があり、パフォーマンスを低下させることなく、ページロード前にユーザーエージェントから Web サーバーに HTTP データを非同期的に転送するためのインターフェイスとしてビーコンを標準化しています。
