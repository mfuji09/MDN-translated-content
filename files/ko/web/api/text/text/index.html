Text() 생성자는 옵션인 파라미터로 주어진 문자형 컨텐츠인 {{domxref("DOMString")}} 으로 새롭게 생성된 {{domxref("Text")}} 객체를 반환합니다.

문법

text1 = new Text(); // 빈 텍스트 노드 생성
text2 = new Text("텍스트 노드입니다");

예제

text = new Text("Test");
