{{APIRef("Performance")}}

摘要

Performance.timing 是唯讀的屬性，傳回的 {{domxref("PerformanceTiming")}} 物件當中包含了效能與時間延遲相關的資訊。

語法

timingInfo = performance.timing;

規格

因為 Navigation Timing 規範已被棄用，此特性不再有望成為標準。請使用 {{domxref("PerformanceNavigationTiming")}} 介面代替。
