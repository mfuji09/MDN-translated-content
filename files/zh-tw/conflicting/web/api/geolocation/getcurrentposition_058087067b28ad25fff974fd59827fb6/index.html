{{APIRef("Geolocation API")}}

是一個正值，它代表機器能夠等待方法回傳位置的最長時間（單位是毫秒）。預設值是Infinity，代表 getCurrentPosition() 此方法在沒有可用的位置前不會有任何回覆。

語法

positionOptions.timeout = timeLength
